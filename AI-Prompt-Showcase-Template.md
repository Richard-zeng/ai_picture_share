# AI 图片提示词作品集

> 这里可以写一句话简介，例如：这个页面用来分享我使用不同提示词生成的 AI 图片作品。

## 内容说明

- 记录不同主题的提示词案例
- 展示生成结果图
- 持续更新我的 AI 图片实验和灵感

## 使用建议

- 每一组案例单独放在一个 `case` 文件夹里
- 图片和 Markdown 文件放在同一个 GitHub 仓库里
- 图片尽量用相对路径引用，这样 GitHub 页面能正常显示

## 建议目录结构

```text
AIpicture/
├─ AI-Prompt-Showcase.md
└─ images/
   ├─ case-01/
   │  ├─ image-01.png
   │  └─ image-02.png
   ├─ case-02/
   │  └─ image-01.png
   └─ case-03/
      └─ image-01.png
```
